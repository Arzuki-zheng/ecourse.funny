### ecourse.funny
It is a fake website come from the students courses web of NUTN
此專案發想於想增進自己對於API開發的能力 包含連接資料庫 資料的整理 和練習前端做串接 
目標是在這個暑假做出一個網站 有網頁 能實作登入 註冊 忘記密碼等一般網頁擁有的功能
